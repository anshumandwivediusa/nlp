# Text Cleaning

## Overview

**Text Cleaning** is the process of removing unwanted, irrelevant, or noisy content from raw text while preserving meaningful information.

Raw text collected from the web, documents, emails, or social media often contains elements that provide little or no value to NLP models. Cleaning the text improves consistency, reduces noise, and enhances the quality of downstream processing.

## Why is it Important?

Real-world text frequently contains:

* HTML tags
* XML tags
* URLs
* Email addresses
* Emojis
* Special characters
* Duplicate whitespace
* Escape characters
* OCR artifacts
* JavaScript or CSS snippets
* Boilerplate content

Removing these elements helps NLP models focus on meaningful language rather than formatting or metadata.



## Example

### Raw Text

```text
<div>Hello 😊</div>

Visit https://example.com

Contact: admin@example.com
```

### Cleaned Text

```text
Hello

Visit

Contact
```

## Common Text Cleaning Operations

| Operation                   | Purpose                                                                    |
| --------------------------- | -------------------------------------------------------------------------- |
| Remove HTML/XML Tags        | Removes markup from web pages and documents                                |
| Remove URLs                 | Eliminates hyperlinks that usually do not contribute semantic meaning      |
| Remove Email Addresses      | Removes personal or system email identifiers                               |
| Remove Special Characters   | Eliminates unnecessary punctuation and symbols                             |
| Remove Emojis               | Removes or replaces emoji characters depending on the task                 |
| Remove Extra Whitespace     | Converts multiple spaces into a single space                               |
| Remove Escape Characters    | Removes characters such as `\n`, `\r`, and `\t` when appropriate           |
| Remove Duplicate Characters | Standardizes repeated characters (e.g., `"soooo"` → `"soo"` or `"so"`)     |
| Remove Boilerplate Text     | Eliminates headers, footers, menus, advertisements, and navigation content |

## Best Practices

* Remove only information that is irrelevant to the NLP task.
* Preserve meaningful punctuation when required.
* Avoid aggressive cleaning for LLMs, as punctuation and formatting may carry semantic information.
* Maintain a consistent cleaning pipeline during both training and inference.
* Validate the cleaned text before tokenization.


## Applications

* Named Entity Recognition (NER)
